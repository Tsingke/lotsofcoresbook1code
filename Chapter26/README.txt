Archive file(s) in this directory dated in 2014 will correspond to 
version(s) used in the book.

To unpack:
   unzip Pearl.zip
   cd Pearl

To build:

  cd to ooc_solver/scalapack
  make

  cd ../../SRC-1mat-MIC-final
  sh run_on_default_intel.sh 

If needed, modify the source commands in run_on_default_intel.sh to point to
 the latest Intel software distribution.
